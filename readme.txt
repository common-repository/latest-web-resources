== Description ==

Daily mix of handy web resources, news, freebies, and tips to make the web experience of your readers fun and easy. Add this widget to your sidebar to show your visitors whats happening of the world of web. This stylish widget will display all the coolest and latest web resources that are buzzing around the web.

== Installation ==

1. Install & activate the plugin
2. Configure the settings in the "Web Resources" option in your WordPress admin panel.
3. Go to the Widgets Section of your WordPress and add the "Web Resources Widget" to your sidebar.
4. Coolest web resources are now available to your readers through your blog.
